## Help

### URL Params
- `id` : Id of the problem to open. `?id=<number>`
- `list` : Show all the loaded problems. `?list=<any value>`

### Key-bindings 
- Show this Help : <kbd>Ctrl</kbd> + <kbd>k</kbd>
- Save Code : <kbd>Ctrl</kbd> + <kbd>s</kbd>
- Hide/Show Problem : <kbd>Ctrl</kbd> + <kbd>h</kbd>
